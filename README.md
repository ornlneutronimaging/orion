# Orion Studio

**A Custom IDE for Neutron Imaging at ORNL**

Orion Studio is a specialized fork of Visual Studio Code, purpose-built for neutron imaging workflows at Oak Ridge National Laboratory's MARS (HFIR) and VENUS (SNS) beamlines. It provides an integrated development environment tailored for neutron data analysis, combining the power of VSCode with domain-specific tools.

## Overview

Orion Studio integrates:
- **Jupyter Notebooks**: Native support for interactive data analysis and visualization
- **AI Assistants**: Intelligent code completion and analysis recommendations
- **Pixi Environment Management**: Seamless Python environment handling
- **Custom Extensions**: Specialized tools for neutron imaging workflows
- **Beamline Integration**: Direct connectivity with MARS and VENUS beamline systems
