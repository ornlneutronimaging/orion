## Guidelines

- **Resolution**: 1200px width recommended for clarity
- **Format**: PNG preferred
- **Theme**: Use Default Light Modern theme for consistency
- **Annotations**: Use red rectangles/arrows to highlight key UI elements
- **Privacy**: Ensure no personal data or credentials are visible
